# IDP-Function-Research
Codebase for the Iterated Distance-to-Prime (IDP) function research, including the counter-example search for d2(n) >=4
This code contains the Python implementation of the Iterated Distance-to-Prime (IDP) search algorithm.

Research Milestone:
This code successfully identified the first integer exception for the d2(n)<=3 bound at n = 20 831 416.
